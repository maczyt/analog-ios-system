# 模拟IOS系统 

> 模拟苹果手机交互行为及其渲染样式

## 暂时交互效果

``` html
1. 开启程序，呈现黑屏效果
2. 可以点击HOME键表示打开手机
3. 输入密码(紧急情况不支持)
4. 通过双击(模拟移动端长按事件)app图标，呈现是否删除app(暂不支持)
5. 可拖拽黑点(后期加入范围监测，尽量和IOS效果统一)
```

---
